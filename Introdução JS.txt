Pesquisar no YT

-Eslint
-Prettier
-CODERUNNER


ESLINT- Analisa esteticamente seu código para encontrar problemas rapidamente. É implementado nas maiorias dos editores de texto e você pode usa-lo como parte do seu processo de integração continua 

PRETTIER- Formatador de código, ou seja, uma ferramenta que faz a formatação nos seus códigos corrigindo aspectos como espaçamentos, virgulas e etc. Resumidamente, essa ferramenta faz com que seu código fique mais bonito, e siga algumas regras de estilo.

CODERUNNER- Esta extensão fornece uma maneira conveniente de executar rapidamente trechos de código ou arquivos inteiros em varias linguagens de programação a partir do vs code (javascript, phyton, java, c++).
